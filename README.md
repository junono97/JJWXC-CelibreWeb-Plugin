# JJWXC-CelibreWeb-Plugin
CelibreWeb之晋江文学城插件
# 用法
 - 搜索 `find / -name metadata_provider`路径。
 - 复制jjwxc_celibre_web_plugin.py文件到metadata_provider/目录下，并注释掉metadata_provider/目录下其他的py文件。
 - 重启项目。
 - 搜索时需要同时输入“书名;作者名”即可。
